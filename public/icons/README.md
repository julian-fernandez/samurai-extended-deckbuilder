# PWA Icons

This directory contains the PWA icons for the L5R Card Search app.

## Required Icons

The following icon sizes are required for full PWA support:

- `icon-16x16.png` - Favicon
- `icon-32x32.png` - Favicon
- `icon-72x72.png` - Android home screen
- `icon-96x96.png` - Android home screen
- `icon-128x128.png` - Android home screen
- `icon-144x144.png` - Windows tiles
- `icon-152x152.png` - iOS home screen
- `icon-192x192.png` - Android home screen
- `icon-384x384.png` - Android splash screen
- `icon-512x512.png` - Android splash screen

## Generating Icons

1. Open `generate-icons.html` in a browser
2. Right-click each generated icon and save as PNG with the appropriate filename
3. Place all PNG files in this directory

## Icon Design

The icons feature:

- Blue gradient background (#3b82f6 to #1d4ed8)
- White card shape with border
- Red Japanese-style circles
- Clean, modern design suitable for app icons

## Fallback

If you don't have the PNG files, the app will still work but may not show custom icons in some contexts.
